# example.hello

Community example plugin for development reference.

| Field | Type | Description | Example |
|-------|------|-------------|---------|
| `enabled` | bool | Enable the plugin | `true` |
| `message` | string | Custom greeting message | `Hello, World!` |

## Example

```yaml
plugins:
  example.hello:
    enabled: true
    message: "Hello, World!"
```
